K8s Monitoring Helm Chart migration utility

This utility can be used to migrate values files from v1 to v2. It's meant to be a best-effort migration utility and might not result in an optimal solution.

For more details about migration, see the migration guide.

This utility will attempt to use the same default values in v1, even though they're not the same defaults in v2.
